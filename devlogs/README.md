# Development Logs

This directory contains daily development logs for the NTUAIS website project.

## Format

Each log file is named by date: `YYYY-MM-DD.md`

## Structure

Each log typically includes:

- **Summary**: Brief overview of the day's work
- **Completed Tasks**: Checklist of finished items
- **Technical Notes**: Important technical decisions or discoveries
- **Next Steps**: Planned work for future sessions
- **AI Collaboration Notes**: When applicable, notes on AI-assisted development
